# superstore-data-warehouse
This project demonstrates a complete Data Warehousing and Business Intelligence solution built using the Superstore dataset. It includes OLAP cube development, multidimensional analysis, and interactive dashboards using Excel and Power BI to support data-driven decision-making.

# Objectives
Design and implement a data warehouse for analytical reporting
Apply OLAP concepts for multidimensional data analysis
Build interactive dashboards for business insights
Analyze sales performance, trends, and key business metrics

# Technology used 
SQL / Database Management System
OLAP Cubes
Microsoft Excel
Power BI
Data Warehousing Concepts (ETL, Star Schema, Fact & Dimension Tables)

#  Key Insights
Identified top-performing product categories and regions
Analyzed seasonal sales trends
Evaluated profit distribution across segments
Provided data-driven recommendations for business improvement
